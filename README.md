# Cite Shop / Pene POS

Offline-first Point of Sale PWA. pnpm monorepo containing the web frontend and the Supabase backend.

## Structure

- `apps/web` — React + Vite PWA frontend
- `supabase` — migrations, edge functions, local config
- `packages` — reserved for future shared code (e.g. cross-platform design tokens)

## Getting started

```bash
pnpm install
pnpm dev
```
